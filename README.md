# Pizzeria Web
Application d'administration de la pizzeria

## Prérequis
* Installer NodeJS et NPM
* Installer grunt-cli
```
npm install -g grunt-cli
```
* Installer bower
```
npm install -g bower
```

## Installation de l'environnement de développement
* Récupérer les dépendances nécessaires
```
npm install
bower install
```
* Lancer l'application
```
grunt serve
```
